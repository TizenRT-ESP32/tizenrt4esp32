examples/lwm2m_test
^^^^^^^^^^^^^^^^^^^
  This is examples to use lwm2m client and server
  For more information, see the examples/lwm2m_test/lwm2m_*_main.c
  or type the command "lwm2m_server help" or "lwm2m_client help" on TASH.

  Configs (see the details on Kconfig):
  * CONFIG_EXAMPLES_LWM2M_TEST

  Depends on:
  * CONFIG_LWM2M_WAKAAMA
  * CONFIG_LWM2M_CLIENT_MODE || CONFIG_LWM2M_SERVER_MODE
